# 1378. Replace Employee ID With The Unique Identifier

## Problem

Write a solution to show the unique ID of each user. If a user does not have a unique ID, show `null`.

Return the result table in any order.

## Example

### Input

#### Employees

| id | name |
| -- | ---- |
| 1 | Alice |
| 7 | Bob |
| 11 | Meir |
| 90 | Winston |
| 3 | Jonathan |

#### EmployeeUNI

| id | unique_id |
| -- | --------- |
| 3 | 1 |
| 11 | 2 |
| 90 | 3 |

### Output

| unique_id | name |
| --------- | ---- |
| null | Alice |
| null | Bob |
| 2 | Meir |
| 3 | Winston |
| 1 | Jonathan |

## SQL Solution

```sql
select e2.unique_id,e1.name 
from Employees e1 left join EmployeeUNI e2
on e1.id = e2.id
```
