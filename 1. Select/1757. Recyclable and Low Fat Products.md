# 1757. Recyclable and Low Fat Products

## Problem
Find the IDs of products that are both low fat and recyclable.

## Example

### Input

| product_id | low_fats | recyclable |
| ---------- | -------- | ---------- |
| 0 | Y | N |
| 1 | Y | Y |
| 2 | N | Y |
| 3 | Y | Y |
| 4 | N | N |

### Output

| product_id |
| ---------- |
| 1 |
| 3 |

## SQL Solution

```sql
SELECT product_id
FROM Products
WHERE low_fats = 'Y'
  AND recyclable = 'Y';
